# Contributing to This Project

Thank you for wanting to improve this project! To contribute, please follow these steps:

## How to Contribute
1. **Fork** the repository.
2. **Clone** your fork locally.
3. Create a new branch: `git checkout -b feature/your-feature-name`.
4. Make your changes. Ensure your scripts pass `shellcheck`.
5. Commit your changes: `git commit -m 'Add some feature'`.
6. Push to your branch: `git push origin feature/your-feature-name`.
7. Open a **Pull Request** against the `main` branch.

## Code Style Guide
* Always include a shebang (e.g., `#!/usr/bin/env bash`) at the top of scripts.
* Use 2 spaces for indentation.
* Quote your variables (e.g., `"$variable"`) to prevent word splitting.
