# My Shell Scripting Playground

A collection of random Bash and Shell scripts created while learning, practicing, and automating tasks. This repository is open-source, and contributions are welcome!

## 📂 Project Structure

*   `basics/` - Fundamental scripts covering variables, loops, and conditionals.
*   `automation/` - Practical scripts for file management, backups, and system checks.
*   `games/` - Fun, interactive terminal-based mini-games.
*   `utilities/` - Quick helper tools for daily terminal use.

## Getting Started

### Prerequisites
Ensure you are using a Unix-like environment (Linux, macOS, or WSL on Windows) with `bash` installed.

### How to Run a Script
1.  **Clone the repository:**
    ```bash
    git clone https://github.com
    cd your-repo-name
    ```
2.  **Make the script executable:**
    ```bash
    chmod +x folder_name/script_name.sh
    ```
3.  **Execute the script:**
    ```bash
    ./folder_name/script_name.sh
    ```

##  Contributing

Got a cool script you wrote while learning? Found a bug in one of mine? Contributions are highly encouraged!

1.  Review the rules in [CONTRIBUTING.md](CONTRIBUTING.md).
2.  Ensure your script includes a proper shebang line: `#!/usr/bin/env bash`.
3.  Run [ShellCheck](https://shellcheck.net) on your code before submitting a pull request to catch syntax errors.
